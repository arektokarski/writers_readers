This project in JAVA is using Threads. The program simulates a queue to the library, with writers and readers standing there. People enter the library one by one. If a writer enters the library, it must be empty, so he waits for others to leave. Once he goes in, no one can go in until he leaves, not even other writers. Readers can enter more than one person, provided that there is no writer there. When they leave, they go to the end of the line. The program runs for a limited time...
